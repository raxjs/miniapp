# rax-plugin-ssr [![npm](https://img.shields.io/npm/v/rax-plugin-ssr.svg)](https://www.npmjs.com/package/rax-plugin-ssr)


`rax-scripts` plugin which make rax app can server-side render pages

## Usage

```json
{
  "plugins": [
    ["rax-plugin-app", {"targets": ["web"]}],
    "rax-plugin-ssr"
  ]
}
```
