# build-plugin-rax-api-builder
Build the compressed dist directory

## Installation

```bash
npm install --save-dev build-plugin-rax-api-builder
```


## Usage
```javascript
// build.json
{
  "plugins": [
    [
      "build-plugin-rax-api-builder",
      {
        "entry": "src/index.ts"
      }
    ]
  ]
}

```
