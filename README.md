<p align="center">
Rax 小程序是基于运行时驱动的完整跨端小程序开发体系。

---

💌 **完整的 Rax DSL：** 使用完整的 Rax（React） DSL，没有任何语法约束

⏱ **兼容 W3C 标准：** 支持标准 DOM 和 BOM API，像开发 Web 应用一样快速开发小程序

✈️ **支持双引擎混用：** 在运行时中，支持局部高性能组件使用编译时方案构建，提供更极致的性能体验

📤 **多端支持：** 完整支持阿里小程序及微信小程序，即将支持字节跳动、百度、快手小程序
